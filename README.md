# 📁 Smart File Organizer Pro

A professional Python + Flask application that automatically organizes files into categorized folders based on file type.

## 🚀 Features

* Scan files in any folder
* Detect file types automatically
* Organize files into:

  * Images
  * Documents
  * Audio
  * Videos
  * Archives
  * Others
* Duplicate filename protection
* Activity log dashboard
* Professional dark-mode UI

## 🛠 Tech Stack

* Python
* Flask
* OS Module
* Shutil Module
* HTML
* CSS

## 📂 Supported File Types

### Images

.jpg, .jpeg, .png, .gif, .webp

### Documents

.pdf, .doc, .docx, .txt, .xlsx, .pptx

### Audio

.mp3, .wav

### Videos

.mp4, .mkv, .avi

### Archives

.zip, .rar

## ▶ Installation

```bash
pip install -r requirements.txt
python main.py
```

Open:

http://127.0.0.1:5000

## 📸 Project Workflow

1. Enter folder path
2. Scan files
3. Detect file types
4. Create category folders
5. Move files automatically
6. Generate activity logs
