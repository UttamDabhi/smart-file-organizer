document.addEventListener('DOMContentLoaded', function () {
  let toggleBtn = document.getElementById('darkModeToggle');
  if (toggleBtn) {
    if (localStorage.getItem('darkMode') === 'enabled') {
      document.body.classList.add('dark-mode');
      toggleBtn.innerText = "☀️";
    }

    toggleBtn.addEventListener('click', () => {
      document.body.classList.toggle('dark-mode');
      if (document.body.classList.contains('dark-mode')) {
        toggleBtn.innerText = "☀️";
        localStorage.setItem('darkMode', 'enabled');
      } else {
        toggleBtn.innerText = "🌙";
        localStorage.setItem('darkMode', 'disabled');
      }
    });
  }
});

કોડ run થાય ત્યારે, બધાં elements already HTML માં present હોવા જોઈએ.

Without this, document.getElementById('darkModeToggle') null હોઈ શકે.

Safety check → જો button present છે તો જ અંદરનો code execute થાય.

4️⃣ if (localStorage.getItem('darkMode') === 'enabled') { ... }

🔹 Full Short Code
document.addEventListener('DOMContentLoaded', () => {
  const toggleBtn = document.getElementById('darkModeToggle');
  if (!toggleBtn) return;

  if (localStorage.getItem('darkMode') === 'enabled') {
    document.body.classList.add('dark-mode');
    toggleBtn.innerText = "☀️";
  }

  toggleBtn.addEventListener('click', () => {
    document.body.classList.toggle('dark-mode');
    const enabled = document.body.classList.contains('dark-mode');
    toggleBtn.innerText = enabled ? "☀️" : "🌙";
    localStorage.setItem('darkMode', enabled ? 'enabled' : 'disabled');
  });
});

1️⃣ document.addEventListener('DOMContentLoaded', () => { ... });

document → Current HTML page DOM (Document Object Model) represent કરે છે.

addEventListener → DOM events (click, load, etc.) listen કરે છે.

DOMContentLoaded → Event fire થાય છે જ્યારે DOM load થઈ જાય, images / CSS load પહેલા.

() => { ... } → Arrow function, anonymous function.

Meaning: Page load થયા પછી જ code run થાય.

2️⃣ const toggleBtn = document.getElementById('darkModeToggle');

const → Variable declaration. Reference change ન થઈ શકે.

toggleBtn → Variable name, button element store કરે છે.

document.getElementById('darkModeToggle') → HTML માં જે element id “darkModeToggle” છે, તે return કરે છે.

Why const? → button reference change થતી નથી, only properties like innerText change થાય.

3️⃣ if (!toggleBtn) return;

if (!toggleBtn) → check કરે છે button exist કરે છે કે નહિ.

return → function exit કરો, code આગળ ન ચાલે.

Reason: Safety measure → button ન હોય તો error ન આવે.

4️⃣ if (localStorage.getItem('darkMode') === 'enabled') { ... }

localStorage → Browser storage, small key-value data save થાય છે.

getItem('darkMode') → Dark mode preference read કરે છે.

=== 'enabled' → Check strict equality (value + type).

Meaning: Page load પર previous dark mode preference apply કરવું.

5️⃣ document.body.classList.add('dark-mode');

document.body → HTML body element.

classList.add('dark-mode') → body માં CSS class add કરે છે.

dark-mode → Custom CSS class, background/text colors change કરે છે.

6️⃣ toggleBtn.innerText = "☀️";

innerText → button text change કરે છે.

"☀️" → Sun icon → Dark mode active → click → switch to light.

7️⃣ toggleBtn.addEventListener('click', () => { ... });

Button પર click event add.

Arrow function run થાય click time.

8️⃣ document.body.classList.toggle('dark-mode');

toggle() → Add/Remove class automatically.

જો class already present → remove

જો class not present → add

9️⃣ const enabled = document.body.classList.contains('dark-mode');

const enabled → new variable, boolean value store.

contains('dark-mode') → check body dark-mode apply છે કે નહિ.

enabled = true → dark mode active

enabled = false → light mode active

🔟 toggleBtn.innerText = enabled ? "☀️" : "🌙";

ternary operator (? :) → condition check → true value / false value.

Dark mode active → "☀️", Light mode → "🌙"

1️⃣1️⃣ localStorage.setItem('darkMode', enabled ? 'enabled' : 'disabled');

Save preference in browser → page reload પછી dark/light mode remember થાય.

setItem(key, value) → key-value pair save.
